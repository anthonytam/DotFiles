CSC 209 tutorial 07 solutions, week of 29 February 2016

Question 2:

Problem:
Add three to the parameter.

Code: 

int f(int x)
{
    int d = x + 3;
    return(d);
}

Solution: 

int f(int x)
{
    return(x + 3);
}

Comment:
Although in this case it's obvious that 'd' is not used later, it's still worth simplifying this and avoiding the unnecessary and meaningless temporary variable. In general, in any place in C in which you can use a variable name to mean its value, you can use a more complex expression instead if useful (and this is true of most programming languages).
