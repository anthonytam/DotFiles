Remember to commit and push this entire directory.
  • Decide what recogniseSomethingRegex recognises. Write this as a Finite State Machine.
  • Decide what recogniseSomethingFSM recognises. Write this as a regular expression.

A well-fed number is an integer in which 7s appear but 9s do not. (Why? Because seven eight nine.)
This means the number can be made up of any digits EXCEPT 9, and must include AT LEAST ONE 7.

  • Modify recogniseWellFedRegex so that it recognises a well-fed number.
  • Modify recogniseWellFedFSM so that it recognises a well-fed number (as defined above).
